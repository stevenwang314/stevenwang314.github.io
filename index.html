About Me

Hello there, let me introduce myself. I am Steven Wang and I have recently graduated from California University of Long Beach in Spring 2019 with a Bachelors with a Computer Engineering degree. While I was in college, I have done several hands-on projects in between both software and hardware and making things happen.

I am currently working at Volunteers of America with my role is to maintain computers and handling an inventory database. I have developed personal skills from work and myself that universally carries over between now and future. I have dealt with configuring computers such as creating emails for users, setting up new computers for users, as well as troubleshooting computers that are not working properly. I also disposed old and broken computers away by destroying hard drives to prevent misuse of data that could leak from elsewhere. I also have an inventory that contains devices that are currently in use or no longer in use. I worked alongside other workers to be able to solve any issues with computers while being able to finish within a deadline.

Outside of work, I have proficiency in C /C++, and JavaScript. I also have some several hands-on experience onto other programming languages out there such as Python, MATLAB, SQL and Verilog. I can use a version source control called Git to be able to see myself and other changes made to files.

I got into C++ from a book from my local library as well as also learning it through a course during college. After becoming proficient in C++, I expanded my horizons and started learning how to incorporate them with APIs. First it started with high-level libraries with SFML/SDL but then after I have participated in the Hackathon hosted by Cal-tech in Fall 2018 by making a C/C++ project with a team, it got me motivated more into lower-level APIs such as the entire Windows OS API, DirectX/DirectWrite API, and Boost as it helps me challenge myself to build a modern-day with a windows, GUI, and essential graphics for programs. I eventually successfully build several programs off of them.

To further expand my own mindset of coding, I then learned Javascript, HTML (Hyper-text markup language), and CSS (Cascading Style Sheets) which not only allows me to build websites, but also allows me to build back-end servers. While learning, I have also created my own website which is this one as a programming exercise. Similar to how I got into C++, I also learned javascript from my library. Eventually, one of my colleagues recommended me a website called freecodecamp.org to learn more about Javascript as well as its libraries that is used to build front and back end. There, it overcame several challenging projects and courses that improved my Javascript more as well as learning valuable libraies along the way.

So far, I have currently learned front end libraries of Bootstrap, JQuery, React, and SASS library, as well as back end with Node.js, Express, Mongoose, and PostgreSQL to build client and server sided apps. While learning PostgreSQL, I also learned the SQL language on how to deal with CRUD (Create, Read, Update, and Delete methods) that I can use to have a powerful database on my skillsets.

Programming is a art of fun, it allows me to build programs that can happen. Although it can be challenging at start but what they always say. It is like a trial-and-error process. My most fun part about programming is intentionally making program crash to be able to learn how to not make it crash when my program is nearly finished. As they always say. “When you don’t succeed, try, try again.”
